Voice Commands

The whole user interface can be interacted with simply by using voice commands. All buttons can be pressed by giving their respective text contents as voice input (for example say "Next" to navigate to the next question). Some examples of additional voice commands can be seen below.
  • "Edit question 2" : Navigates to the question with the given number. Usable when the review form page is open.
  • "Voice command 2": Lorem ipsum, dolor sit amet consectetur adipisicing elit. Placeat ipsum quae sapiente reprehenderit porro debitis.
  • "Voice command 3": Lorem ipsum dolor, sit amet consectetur adipisicing elit. Odio, magnam.
